# SubTrack

SubTrack is a full-stack subscription management web application.

## Features
- User authentication with JWT
- Add, edit, and delete subscriptions
- Monthly, yearly, and custom billing
- Category-based analytics
- Renewal alerts and notifications
- Savings and usage tracking

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: PostgreSQL

## Project Structure
- `frontend/` - React app
- `backend/` - Express API and database logic
